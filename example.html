You will use the javascript file to continually change the size of the header,bur first we need learn how use JavaScript to actually grab the HTML object form the DOM.

Here are some important documents attributes:

  • document.URL -- This is the actual URL of the website
  • document.body -- This is everything inside of the body
  • document.head -- This everthing in the head of the page
  • document.links -- These are all the links on the page

Then we also have methods we can use to grab HTML elements:

  • document.getElementById() -- Returns the element with the id
  • document.getElementByClassName() -- Returns list of all elements belonging to a class
  • document.getElementByTagName() -- Returns list of all elements with tag
  • document.querySelector() -- Returns the first object matching the CSS style selector
  • document.querySelectorAll() -- Return all object matching the CSS style selector

The first three of these are pretty self explanatory. The querySelector() can accept CSS style selector. so you can use# or if you want id's or class. Basically a more generalized method where you can type any CSS style. The difference between querySelector() and querySelectorAll() is that querySelectorAll() will return list of all matches. querySelector() just returns the first match.
